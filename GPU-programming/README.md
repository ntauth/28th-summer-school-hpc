# GPU programming

## Slides

The slides are available in PDF format
```bash
cd slides
```

## Exercises

A skeleton of exercises are provided

#### OpenACC exercises

```bash
cd hands-on/openacc
```

#### CUDA exercises

```bash
cd hands-on/cuda
```

At the end of course the solutions of all exercises are published in the repository

#### GNU environment
